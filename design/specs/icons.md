# 图标系统设计规范

## 图标风格

- **风格**：线性图标，2px 线宽
- **圆角**：轻微圆角，保持友好感
- **颜色**：主绿色 (#4CAF50) 或中性色
- **尺寸**：16px, 20px, 24px, 32px

## 图标使用规范

### 尺寸规范

- **导航栏图标**：24px
- **按钮图标**：20px
- **列表图标**：20px
- **卡片图标**：24px
- **大图标**：32px（空状态、加载）

### 颜色规范

- **主要图标**：主绿色 (#4CAF50)
- **次要图标**：中性色 (#616161)
- **禁用图标**：浅灰色 (#9E9E9E)
- **功能图标**：根据功能使用对应颜色

### 间距规范

- **图标与文字**：8px
- **图标之间**：12px

## 图标库建议

### 推荐图标库

1. **Iconify** - 丰富的图标库
2. **Feather Icons** - 简洁线性图标
3. **Material Icons** - 谷歌 Material Design
4. **自定义图标** - 植物相关图标可自定义

### 自定义图标设计

#### 植物图标
```
风格：简化的植物轮廓
特点：识别度高、简洁
颜色：主绿色渐变
```

#### 养护图标
```
风格：圆形背景 + 图标
特点：统一风格、易于识别
背景：浅绿色 (rgba(76, 175, 80, 0.1))
```

## 图标动画

### 加载动画

```css
/* 旋转动画 */
@keyframes rotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

.rotate-icon {
  animation: rotate 1s linear infinite;
}
```

### 点击反馈

```css
/* 缩放动画 */
.icon-button:active {
  transform: scale(0.95);
  transition: transform 0.1s;
}
```

## 图标文件格式

- **SVG**：优先使用，矢量图，可缩放
- **PNG**：备用方案，需提供 @2x 和 @3x
- **字体图标**：适合简单图标

## 图标命名规范

```
功能_状态_尺寸.svg

示例：
- camera_default_24.svg
- search_active_20.svg
- plant_empty_32.svg
```
