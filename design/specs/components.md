# 组件设计规范

## 按钮组件

### 主要按钮（Primary Button）

用于主要操作，如"识别植物"、"加入我的花园"。

```
┌─────────────────────┐
│   识别植物           │
└─────────────────────┘

样式：
- 背景：主绿色 (#4CAF50)
- 文字：白色 (#FFFFFF)
- 圆角：8px
- 内边距：12px 24px
- 字体：16px, 500
- 高度：44px（最小触摸区域）
- 阴影：0 2px 8px rgba(76, 175, 80, 0.3)

状态：
- 默认：主绿色背景
- 悬停：浅绿色 (#66BB6A)
- 按下：深绿色 (#388E3C)
- 禁用：灰色 (#E0E0E0)，文字 #9E9E9E
```

### 次要按钮（Secondary Button）

用于次要操作，如"取消"、"重新选择"。

```
┌─────────────────────┐
│   重新选择           │
└─────────────────────┘

样式：
- 背景：白色 (#FFFFFF)
- 文字：主绿色 (#4CAF50)
- 边框：1px solid #4CAF50
- 圆角：8px
- 内边距：12px 24px
- 高度：44px
```

### 文字按钮（Text Button）

用于辅助操作，如"结果不准？"。

```
样式：
- 背景：透明
- 文字：主绿色 (#4CAF50)
- 下划线：可选
- 字体：14px
```

## 卡片组件

### 标准卡片

用于展示植物信息、养护要点等。

```
┌─────────────────────────────┐
│                             │
│   [植物图片]                 │
│                             │
│   植物名称                   │
│   学名                       │
│   养护要点...                │
│                             │
└─────────────────────────────┘

样式：
- 背景：白色 (#FFFFFF)
- 圆角：12px
- 内边距：20px
- 阴影：0 2px 8px rgba(0, 0, 0, 0.1)
- 边框：无（或 1px #E0E0E0）
```

### 植物卡片

用于"我的花园"列表。

```
┌─────────────────────────────┐
│  [图片]  植物名称            │
│           学名               │
│          下次浇水：3天后     │
│          [操作按钮]          │
└─────────────────────────────┘

样式：
- 背景：白色
- 圆角：12px
- 内边距：16px
- 间距：16px（卡片之间）
```

## 输入框组件

### 标准输入框

```
┌─────────────────────────────┐
│ 输入植物名称...              │
└─────────────────────────────┘

样式：
- 背景：白色 (#FFFFFF)
- 边框：1px solid #E0E0E0
- 圆角：8px
- 内边距：12px 16px
- 字体：16px
- 高度：44px

状态：
- 默认：灰色边框
- 聚焦：主绿色边框 (#4CAF50)
- 错误：红色边框 (#E57373)
```

### 搜索框

```
┌─────────────────────────────┐
│ 🔍 输入植物名称搜索...       │
└─────────────────────────────┘

样式：
- 左侧图标：搜索图标
- 右侧：清除按钮（有内容时显示）
```

## 提示组件

### Toast 提示

```

│ ✓ 已添加到我的花园  │
└─────────────────────┘

样式：
- 背景：rgba(0, 0, 0, 0.8)

```

### 对话框

```
┌─────────────────────────────┐
│  确认删除？                  │
│                              │
│  删除后无法恢复               │
│                              │
│  [取消]  [确认删除]          │
└─────────────────────────────┘

样式：
- 背景：白色
- 圆角：12px
- 宽度：80%（最大 400px）
- 阴影：0 4px 16px rgba(0, 0, 0, 0.2)
```
