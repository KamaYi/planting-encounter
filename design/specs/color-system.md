# 色彩系统设计规范

## 主色调 - 绿色系

### 主绿色（Primary Green）

植遇应用的核心色彩，代表生命力、自然和成长。

| 色值 | 用途 | 说明 |
|------|------|------|
| `#4CAF50` | 主色 | 标准绿色，用于主要按钮、强调元素 |
| `#66BB6A` | 主色-浅 | 悬停状态、次要按钮 |
| `#388E3C` | 主色-深 | 按下状态、重要提示 |
| `#2E7D32` | 主色-更深 | 深色模式下的主色 |

**使用场景**：
- 主要操作按钮（识别、添加、确认）
- 导航栏背景
- 选中状态
- 进度条、加载动画
- 成功提示

### 辅助绿色（Secondary Green）

用于补充和层次区分。

| 色值 | 用途 | 说明 |
|------|------|------|
| `#81C784` | 浅绿 | 背景色、卡片边框 |
| `#A5D6A7` | 极浅绿 | 浅色背景、分割线 |
| `#C8E6C9` | 最浅绿 | 页面背景、输入框背景 |

### 功能色（Functional Colors）

| 颜色 | 色值 | 用途 |
|------|------|------|
| 成功 | `#4CAF50` | 操作成功、完成状态 |
| 警告 | `#FFB74D` | 提醒、注意事项 |
| 错误 | `#E57373` | 错误提示、删除操作 |
| 信息 | `#64B5F6` | 信息提示、链接 |

### 中性色（Neutral Colors）

用于文字、背景和分割线。

| 色值 | 用途 | 说明 |
|------|------|------|
| `#FFFFFF` | 纯白 | 卡片背景、页面背景 |
| `#F5F5F5` | 浅灰 | 页面背景、输入框背景 |
| `#E0E0E0` | 中灰 | 分割线、边框 |
| `#9E9E9E` | 深灰 | 次要文字、占位符 |
| `#616161` | 更深灰 | 次要标题 |
| `#212121` | 最深灰 | 主要文字、标题 |

## 色彩搭配示例

### 卡片设计
```
背景：白色 (#FFFFFF)
边框：浅绿色 (#A5D6A7, 1px)
阴影：rgba(76, 175, 80, 0.1)
```

### 按钮设计
```
主要按钮：
- 背景：主绿色 (#4CAF50)
- 文字：白色 (#FFFFFF)
- 悬停：浅绿色 (#66BB6A)
- 按下：深绿色 (#388E3C)

次要按钮：
- 背景：白色 (#FFFFFF)
- 文字：主绿色 (#4CAF50)
- 边框：主绿色 (#4CAF50, 1px)
```

### 状态指示
```
待处理：浅灰色 (#E0E0E0)
进行中：主绿色 (#4CAF50)
已完成：深绿色 (#2E7D32)
警告：橙色 (#FFB74D)
错误：红色 (#E57373)
```

## 渐变设计

### 页面头部渐变
```css
background: linear-gradient(180deg, #4CAF50 0%, #66BB6A 50%, #F5F5F5 100%);
```

### 卡片渐变
```css
background: linear-gradient(135deg, #FFFFFF 0%, #F1F8E9 100%);
```

## 透明度使用

| 透明度 | 用途 |
|--------|------|
| 10% | 浅色背景、分割线 |
| 20% | 遮罩层、加载状态 |
| 50% | 禁用状态 |
| 80% | 半透明卡片 |

## 色彩对比度

确保文字与背景的对比度符合 WCAG AA 标准：

- 主要文字：至少 4.5:1
- 大号文字：至少 3:1
- 交互元素：至少 3:1
