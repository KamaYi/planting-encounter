# 页面设计规范

## 1. 首页 - 植物识别

### 页面结构

```
┌─────────────────────────────┐
│  [导航栏 - 主绿色背景]       │
│  植遇                        │
├─────────────────────────────┤
│                              │
│       植物识别                │
│   拍照或上传图片，识别植物    │
│       并获取养护指南          │
│                              │
│   ┌─────────────────────┐   │
│   │                     │   │
│   │   📷 点击拍照或      │   │
│   │   选择图片          │   │
│   │                     │   │
│   └─────────────────────┘   │
│                              │
│   💡 提示：请确保图片清晰     │
│   🔍 识别结果不准确？可以     │
│      手动纠正                │
│                              │
└─────────────────────────────┘
```

### 设计要点

- **头部渐变**：从主绿色渐变到浅灰色背景
- **上传区域**：大尺寸卡片，圆角 16px，浅绿色边框
- **提示信息**：浅色背景卡片，友好提示
- **按钮**：主要按钮使用主绿色

### 交互流程

1. 用户点击上传区域
2. 弹出选择：拍照 / 相册
3. 选择图片后显示预览
4. 自动开始识别（显示加载动画）
5. 识别完成后跳转详情页

## 页面通用规范

### 导航栏

- **背景**：主绿色 (#4CAF50)
- **文字**：白色 (#FFFFFF)
- **高度**：44px（iOS）或 48px（Android）
- **返回按钮**：左侧，白色图标

### 页面背景

- **主背景**：浅灰色 (#F5F5F5)
- **卡片背景**：白色 (#FFFFFF)

### 间距规范

- **页面内边距**：20px
- **卡片间距**：16px
- **元素间距**：12px
